## Authority

The AURA Constitution is the highest-authority document in the entire Aura Protocol hierarchy. Every other document, invariant, implementation, and decision must be consistent with it.

## Amendment

Amendments require: RFC → Architecture Review → Impact Analysis → Dependent Document Updates → Chief Architect Approval.

A FROZEN version cannot be changed. Amendments create a new version.
