# Glossary

This directory provides the canonical glossary for the Aura Protocol, supplementing APS-000.

## Documents

| File | Description |
|------|-------------|
| [GLOSSARY.md](GLOSSARY.md) | Alphabetical index of all canonical terms |

## Usage

Always use terms from this glossary in specification documents, ADRs, RFCs, and implementation documentation. Using non-canonical synonyms is a documentation defect.

See [../aps/APS-000_FOUNDATION_AND_TERMINOLOGY.md](../aps/APS-000_FOUNDATION_AND_TERMINOLOGY.md) for the authoritative term definitions.
