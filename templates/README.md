# Templates

Authoring templates for all document types in `aura-specification`.

## Available Templates

| Template | Use For |
|----------|---------|
| [ADR_TEMPLATE.md](ADR_TEMPLATE.md) | Architecture Decision Records |
| [RFC_TEMPLATE.md](RFC_TEMPLATE.md) | Requests for Comments |
| [CONFORMANCE_TEST_TEMPLATE.md](CONFORMANCE_TEST_TEMPLATE.md) | Conformance Test definitions |
| [FIXTURE_TEMPLATE.json](FIXTURE_TEMPLATE.json) | Reference Fixtures |
| [CONFORMANCE_REPORT_TEMPLATE.md](CONFORMANCE_REPORT_TEMPLATE.md) | Implementation Conformance Reports |
| [APS_DOCUMENT_TEMPLATE.md](APS_DOCUMENT_TEMPLATE.md) | New APS specification documents |

## Usage

1. Copy the appropriate template
2. Replace all `[PLACEHOLDER]` values
3. Remove any sections not applicable
4. Follow the [STYLE_GUIDE.md](../STYLE_GUIDE.md)
