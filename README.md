# Aura Specification

Repozytorium zawiera kompletną dokumentację specyfikacji Aura, w tym:

- **APS (Aura Protocol Specification)** – opis standardu, architektury i wymagań technicznych.
- **Constitution** – zasady ładu projektu, role, procesy decyzyjne i reguły zmian.

## Struktura dokumentacji

- `/docs/APS.md` – pełna specyfikacja APS
- `/docs/CONSTITUTION.md` – pełna Constitution projektu

## Wersjonowanie dokumentacji

Aktualna wersja: **v1.0.0**

Wszystkie zmiany w dokumentacji powinny:

1. zachowywać spójność między APS i Constitution,
2. aktualizować numer wersji przy zmianach normatywnych,
3. zawierać krótkie uzasadnienie zmiany w opisie commita/PR.
