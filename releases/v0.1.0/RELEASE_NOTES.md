## Summary

Initial release of the `aura-specification` repository. This release establishes the canonical documentation structure for the Aura Protocol.

---

## What's Included

- AURA Constitution v1.0 (FROZEN) in canonical Markdown
- APS-000 through APS-950 as canonical Markdown documents (all DRAFT)
- Protocol Invariant Registry (INV-001 – INV-015)
- Conformance Test stubs (CONF-001 – CONF-010)
- Reference Fixture placeholder (FIX-001)
- Traceability Model and Matrix
- Governance documents (GOVERNANCE.md, CONTRIBUTING.md, VERSIONING.md, STYLE_GUIDE.md)
- ADR-001 documenting the repository structure decision
- Templates for ADRs, RFCs, Conformance Tests, Fixtures, Conformance Reports
- Reference Implementation status documents (RI-PY, RI-RS)

---

## Known Gaps

- APS-001 does not exist (highest priority)
- All APS documents are DRAFT status
- Canonical fixture data pending APS-200 schema finalization
- 5 invariants lack Conformance Tests (INV-007, INV-012, INV-013, INV-014, INV-015)
- No implementations are currently certified

See [ROADMAP.md](../../ROADMAP.md) for remediation schedule.

---

## Compatibility

No protocol conformance certification is possible for this release. This release establishes the specification structure only.
